Morfología del Robot: Actuadores, Sensores Internos y Elementos Terminales

1. ThomasNet

ThomasNet, Platafotma en línea para la busqueda de proveedores para industrias en USA y Canada definen los actuadores y habla de otros puntos:

  • "es un tipo de herramienta utilizada para hacer que algo lleve a cabo una acción de forma automática"
  • "Hay muchas razones por las cuales se utilizan actuadores para realizar trabajos. Muchos de estos son utilizados para situar o posicionar un objeto en particular. Tambien pueden ser utilizador para movilizar aparatos mecánicos o herramientas que llevan a cabo un trabajo."
  • " Al utilizar actuadores en un área de trabajo se facilita trabajar en este espacio y se facilita el uso de ciertas herramientas."

Fuente: [ https://www.thomasnet.com/about/actuators-301168.html ]

3. Elementos Terminales (End Effectors o Grippers)

En el libro Robot Grippers, Monkman define a los elementos terminales como "el aparato colocado al final de una extremidad robótica diseñado para interactuar con su entorno. "

  • "Actuadores Hidráulicos: Se utiliza mecánica de Fluidos para operarlos"
  • "Actuadores Neumáticos: Utiliza gases para operalos"
  • "Actuadores Electricos, TCP, Magnético, etc // investigar más"

Fuente: [ Monkman, G. J.; Hesse, S.; Steinmann, R.; Schunk, H. (2007). Robot Grippers. Wiley-VCH. p. 62. ISBN 978-3-527-40619-7.]
